# **4) Practical “what your devs script next”**  
With the above, scripting becomes deterministic:  
## **A) Parse Canva design names**  
```
{surface}__{asset_class}__{asset_id}__{aspect}__v{n}

```
## **B) Derive fit automatically**  
* fit.surfaces = [surface]  
* fit.altitudes can be inferred from surface prefix (cmp_, con_, cc_, mkt_, sys_)  
* fit.rooms can be inferred by mapping surface → room (they’ll maintain a SURFACE_TO_ROOM dictionary)  
## **C) Transcode + upload**  
* Canva exports: png/jpg/mp4  
* Pipeline transcodes: avif/webp/webm + posters  
* Upload to /assets/{asset_id}/{aspect}/{width}.{ext}  
## **D) Registry upsert**  
* Same asset_id + higher version wins  
* Old versions remain immutable (or get “deprecated”)
