## Usage Guidelines

### Tag Structure
- `intent.*` - Primary purpose (growth, healing, focus, calm, etc.)
- `context.*` - Usage context (meditation, recovery, productivity, etc.)
- `tone.*` - Emotional tone (gentle, empowering, peaceful, etc.)
- `length.*` - Text length (short: <15 chars, medium: 15-30 chars)

### Selection Logic
1. Match primary intent + context
2. Fallback to related intents if no exact match
3. Consider tone appropriateness for user state
4. Prefer medium length for most use cases

### A/B Testing Notes
- Track engagement metrics by copy variation
- Test tone effectiveness across user segments
- Monitor copy performance by therapeutic style pairing
