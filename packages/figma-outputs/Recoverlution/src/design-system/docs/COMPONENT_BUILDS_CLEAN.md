## Glossary (drop near first use)
- KBE: Knowing, Believing, Embodying.
- Navicue: Guided micro-intervention with voice and timer.
- Receipt: Proof artifact with a measurable delta.
- Transfer test: Repetition of the win in a new context.
- Archetype: Routing profile combining heat, schema pattern, and default primitives.
- Mindblock: If-X-then-Y pattern we aim to rewrite.

## Tone rules
- Plain, precise, warm. No hype words (amazing, magic). Use short sentences.
- Prefer verb-first CTAs. Keep casing in Title Case for headings, sentence case for body.
- Keep numbers concrete (e.g., "8 families", "90-second timer", "delta -4").
